- Go to https://github.com/bible-technology/scribe-scripture-editor/releases/
- To download click the **Scribe** program that matches your computer's Operating System (OS) 
  - Supported Operating System 
    - Windows <img src="/autographaAssetsV2/getting-started/windows.png" alt="Windows Icon" width="50px"/>
    - Mac <img src="/autographaAssetsV2/getting-started/mac-os.png" alt="MacOS Icon" width="50px"/>
    - Ubuntu <img src="/autographaAssetsV2/getting-started/linux.png" alt="Linux Icons" width="50px" />
- Right-click on the file to open the menu option
- On the menu option, select **Run** or **Open**

**Steps to Install Scribe on Windows OS**
- Double-click the application file (.exe) you've downloaded from the GitHub
-  Follow the instruction until installed

Windows OS restrictions: If you receive a pop-up screen with the message, “Windows protected your PC,” perform the following actions. 

- Click on the **More Info** button
- Click the **Run Anyway** button at the bottom of the window
- Choose Installation options **Only for me**
- Follow the instructions in the installation window until **Finish** appears
- After installing the application, you can use the **Scribe* shortcut automatically created for you on the Desktop to access the application

Some devices may prevent the installation of **Scribe** because of operating system restrictions. 

**Steps to Install Scribe on Mac OS**

- Double-click the application file (.dmg) you've downloaded from the GitHub
- Drag the application to the Applications folder
- Open the application from the Application launcher

Mac OS restrictions: If you receive a pop-up screen with the message, **Scribe** cannot be opened because the developer cannot be verified,” perform the following actions. 

- Go to **Finder**
- Click on **Applications**
- Locate the **Scribe** application
- **Right-Click** or press the **control key** and click on the **Scribe** icon
- Click **Open**
- A pop-up message will appear on the screen saying, “macOS cannot verify the developer of “Scribe”. Are you sure you want to open it?”
- Click **Open** on the pop-up
- Follow the instructions in the installation window until **Finish** appears

 **Steps to Install Scribe on Ubuntu OS**

- Locate the downloaded deb package. By default, the system stores downloaded files in the Downloads directory
- Right-click the deb file and choose Open with Another Application
- Select Software Install from the program list
- A new dialogue box opens with details about the software
- Click on the Install button to proceed
- Enter the user's password when prompted and press Enter to authenticate
- Wait for the installation to complete. When finished, the software is ready to use
